# Qu'est-ce qu'un terminal ?

Un **terminal** est un outil qui permet d'interagir avec le système
d'exploitation de votre ordinateur en utilisant des commandes textuelles. Il
est souvent utilisé pour exécuter des programmes, gérer des fichiers et
effectuer diverses tâches administratives. Il évite d'avoir à utiliser une
interface graphique (GUI) pour effectuer ses tâches.

Dans ce cours, nous allons principalement utiliser le terminal pour exécuter des
programmes Python et utiliser Git.

## Où est mon terminal ?

Sur Windows, vous pouvez utiliser **PowerShell**. Sur
Mac, il s'appelle **Terminal**. Sur Linux, il peut s'appeler **Terminal** ou
**Konsole** selon la distribution.

## Terminal dans VS Code

Dans VS Code, vous pouvez ouvrir le terminal intégré en allant dans le menu
`Affichage > Terminal`. Le terminal se trouve dans le panneau inférieure de
l'éditeur. Il existe un raccourci clavier pour ouvrir cette fenêtre, soit
`Ctrl + J` ou `Cmd + J` sur Mac.
