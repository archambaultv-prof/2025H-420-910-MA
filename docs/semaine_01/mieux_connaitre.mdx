# Questionnaire pour mieux vous connaître

Dans le but de mieux vous connaître et adapter mon enseignement, je vous
demande de répondre aux questions suivantes.

## Comment répondre?

Puisqu'il s'agit d'un cours de programmation, vous devez répondre aux questions
dans VS Code ou un autre éditeur de texte.

Voici comment faire :

1. Créer un fichier texte avec le nom `questionnaire-VOTRE-NOM.txt` sur votre ordinateur.
2. Copier les questions ci-dessous dans le fichier texte.
3. Répondre aux questions en ajoutant vos réponses après chaque question.
4. Enregistrer le fichier texte et le déposer sur Omnivox, dans la section Travaux et remises.

## Questions

*Vous pouvez sauter une question si vous la juger trop personnelle*

### Question générale

- Pourquoi avoir choisi un l'AEC Développement de logiciels - Sécurité d'applications de bureau, mobiles et Web
  au Collège de Maisonneuve ? 

- Avez-vous déjà une idée du type d'emploi ou de projet que vous aimeriez faire après votre AEC ?

- Est-ce que vous travaillez en même temps que vous étudiez ? Si oui, combien d'heures par semaine environ ?

- Avez-vous des responsabilités familiales ?

- Quels sont vos passe-temps ou centres d’intérêt en dehors de l’informatique ?


### Question en lien avec le cours

- Avez-vous déjà des connaissances en programmation ? Si oui, donnez
  quelques exemples de ce que vous savez faire.

- Avez-vous déjà utilisé un ordinateur pour autre chose que naviguer sur le Web
ou écrire des documents ? Si oui, donnez quelques exemples (ex. : Excel, jeux,
montage vidéo, Linux, etc.)

- Quelle est votre aisance avec l’anglais écrit (lecture de documentation
  technique, messages d’erreur, tutoriels) ?

- Est-ce que vous avez déjà utilisé des outils de développement comme Visual
  Studio Code, GitHub ou autre ?
