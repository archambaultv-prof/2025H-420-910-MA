# Exercices de la semaine 1

## VS Code

Apprendre à utiliser VS Code comme un outil de travail efficace pour
programmer, organiser ses fichiers et collaborer.

<ExerciseList>

<Exercise>
  <ExerciseHeader>
  ### Créer un dossier pour les exercices
  </ExerciseHeader>
  <ExerciseBody>

    - Ouvrir VS Code.
    - Créer un nouveau dossier. Par exemple `exercices-semaine-01`.
    - Créer à l’intérieur :
        - un fichier `algorithmes.txt`
    - Ajoute un commentaire dans le fichier : `Mon premier algorithme`

   </ExerciseBody>
</Exercise>

</ExerciseList>

## Algorithme et pseudo-code

Écrire dans le fichier `algorithmes.txt` un algorithme pour les exercices suivants. Vous pouvez utiliser un langage naturel
ou du pseudo-code.

<ExerciseList>

<Exercise>
  <ExerciseHeader>
  ### Plus petit de trois nombres
  </ExerciseHeader>
  <ExerciseBody>
  Écrivez un algorithme qui lit trois nombres et affiche le plus petit des trois.
   </ExerciseBody>
</Exercise>

<Exercise>
  <ExerciseHeader>
  ### Vérification d’année bissextile
  </ExerciseHeader>
  <ExerciseBody>

  Rechercher les critères pour déterminer si une année est bissextile.
  
  Écrire un algorithme qui lit une année et affiche si elle est bissextile ou
  non.
  
   </ExerciseBody>
</Exercise>

<Exercise>
  <ExerciseHeader>
  ### Distribution de boîtes
  </ExerciseHeader>
  <ExerciseBody>

  Une entreprise doit emballer des produits dans des boîtes. Chaque boîte peut
contenir 12 produits. Écrivez un algorithme qui :
  - Lit le nombre total de produits à emballer
  - Calcule combien de boîtes complètes sont nécessaires
  - Affiche combien de produits seront dans la dernière boîte (s’il en reste)

   </ExerciseBody>

</Exercise>

</ExerciseList>
