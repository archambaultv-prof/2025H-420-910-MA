# Retour TP 1

Bien réussi dans l'ensemble. Voici quelques remarques :

- Pour une évaluation sommative, mieux vaut ne pas toucher au code de l'énoncé, sauf si explicitement demandé.
- Une fois que le parseur à fini son travail, il n'est plus nécessaire. Le reste se fait sur le code AST (abstract syntax tree).
- Testez bien votre code pour s'assurer qu'il fonctionne avant la remise.
